# 🎯 Jogo de Adivinhação

Projeto simples desenvolvido para praticar lógica de programação com JavaScript.

## 🚀 Funcionalidades
- Geração de número aleatório
- Sistema de tentativas
- Feedback de "maior" ou "menor"
- Game Over com limite de tentativas
- Reinício do jogo

## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript
